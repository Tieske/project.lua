do_something (what[, force=false])
Does something. It will do what you tell it.

Parameters:

  • what string or function the thing that has to be done
  • force boolean set to a truthy value to force it. (default false)

Returns:

  1. boolean success
  2. nil or string error message on failure

Usage:

    local success, err = project.do_something("tell a lie", true)
if not success then
    print("failed at lying; ", err)
end
